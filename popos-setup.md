#### Get Tibetan working
Add iBus m17n:
```bash
sudo apt install ibus-m17n
```
Will need to restart.

To configure iBus input methods:
```bash
ibus-setup
```
#### Install flatpack
```bash
sudo apt install flatpak
flatpak remote-add --user --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo
```
May need to restart?
